# Image Translation using OpenCV Python

## Overview
This project demonstrates Image Translation using Python and OpenCV.  
Image translation is a geometric transformation technique used to move an image from one position to another along the x-axis and y-axis.

The project is implemented using OpenCV’s affine transformation functions and helps beginners understand the basics of image processing and computer vision.

---

## Features
- Translate images horizontally and vertically
- Perform affine transformation using OpenCV
- Beginner-friendly Python implementation
- Simple and easy-to-understand code
- Supports different image formats

```bash

```

```bash

```

---

## How to Run

Run the Python file using:

```bash
python image_translation.py
```

---

## How It Works

1. The input image is loaded using OpenCV.
2. A translation matrix is created.
3. The image is shifted along the x-axis and y-axis.
4. OpenCV’s `warpAffine()` function applies the transformation.
5. The translated image is displayed and saved.
